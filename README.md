# IPL Performance Analytics 

## Project Overview

This project analyzes IPL match and ball-by-ball data to generate business insights using Python, SQL, and Power BI.

The dashboard helps answer questions such as:

* Which teams are most successful?
* Does winning the toss affect match results?
* Who are the best batsmen and bowlers?
* Which venues are batting-friendly or bowling-friendly?
* Who are the best finishers and powerplay batsmen?
* Which players provide the highest overall value (MVP Score)?

## Tools Used

* Python
* Pandas
* NumPy
* SQL
* Power BI
